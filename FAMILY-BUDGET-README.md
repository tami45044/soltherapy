# 🏠 Family Budget - מערכת ניהול תקציב משפחתי

## 📋 תיאור
מערכת מקיפה לניהול הוצאות והכנסות של משק הבית עם ויזואליזציות וסיכומים מפורטים.

## 🚀 גישה למערכת
כתובת URL: `/family-budget`

המערכת מוגנת באימות Firebase - נדרשת כניסה עם אימייל וסיסמא.

## ✨ תכונות עיקריות

### 1. דשבורד ראשי
- סקירה כללית של המצב הכלכלי החודשי
- כרטיסים עם סה"כ הכנסות, הוצאות ויתרה
- גרפים אינטראקטיביים:
  - חלוקת הוצאות לפי קטגוריה (עוגה)
  - חלוקת הוצאות לפי סוג (דונאט)
- פעולות מהירות לכל החלקים של המערכת

### 2. הזנה מהירה
- טופס מהיר להוספת הוצאות והכנסות
- מעבר קל בין הוצאה להכנסה
- שדות עם ולידציה מלאה
- ברירת מחדל לתאריך היום

#### אמצעי תשלום:
- **אשראי** - תשלום בכרטיס אשראי
- **מזומן** - תשלום במזומן
- **הוראת קבע** - תשלום בהוראת קבע
- **העברה בנקאית** - העברה מחשבון הבנק

#### קטגוריות הוצאות:
- מזון ומשקאות
- דלק ותחבורה
- בילויים ופנאי
- ביגוד והנעלה
- בריאות ותרופות
- חינוך וספרים
- קניות ושונות
- משכנתא
- ארנונה
- חשמל
- מים
- גז
- ביטוח
- טלפון ואינטרנט
- מנויים
- אחר

#### קטגוריות הכנסות:
- משכורת תלוש
- העברה מעסק
- תשלום מלקוח
- קיצבת ילדים
- העברה ממשפחה
- אחר

### 3. ניהול הוצאות
- טבלה מפורטת של כל ההוצאות
- סינון לפי:
  - חודש
  - סוג הוצאה
  - קטגוריה
  - חיפוש חופשי
- עריכה ומחיקה של הוצאות קיימות
- סה"כ דינמי לפי הסינון

### 4. ניהול הכנסות
- טבלה מפורטת של כל ההכנסות
- סינון לפי:
  - חודש
  - קטגוריה
  - חיפוש חופשי
- עריכה ומחיקה של הכנסות קיימות
- סה"כ דינמי לפי הסינון

### 5. סיכום חודשי
- כרטיסים עבור כל חודש עם:
  - סה"כ הכנסות והוצאות
  - יתרה (עודף/גירעון)
  - 3 הקטגוריות המובילות
- מיון לפי חודש (האחרון ראשון)
- אפשרות לצפייה מפורטת

### 6. סיכום שנתי
- בחירת שנה לצפייה
- כרטיסי סיכום:
  - סה"כ הכנסות שנתי
  - סה"כ הוצאות שנתי
  - יתרה שנתית
  - ממוצע הוצאות חודשי
- גרפים מתקדמים:
  - **מגמה חודשית** - גרף קו של הכנסות והוצאות לאורך השנה
  - **השוואת הכנסות והוצאות** - גרף עמודות
  - **יתרה חודשית** - גרף עמודות מצבעי

### 7. הוצאות קבועות (תבניות)
- הגדרת הוצאות חוזרות חודשיות
- דוגמאות: משכנתא, ארנונה, חשמל, מים
- אפשרות להשהות/להפעיל הוצאה קבועה
- סה"כ הוצאות קבועות פעילות
- כרטיסי תצוגה נוחים

## 🗄️ מבנה נתונים

### Collections ב-Firestore:
1. **familyBudgetExpenses** - הוצאות
2. **familyBudgetIncome** - הכנסות
3. **familyBudgetFixedExpenses** - תבניות הוצאות קבועות

## 📊 טכנולוגיות
- **Vue 3** + Composition API
- **Vuetify 3** - UI Framework
- **Firebase Firestore** - מסד נתונים
- **Chart.js** - גרפים ווויזואליזציות
- **TypeScript** - בטיחות טיפוסים

## 🎨 עיצוב
המערכת משתמשת בערכת צבעים ירוקה (green theme) להבדלה ממערכת הניהול הקיימת:
- ירוק - הכנסות
- אדום - הוצאות
- כחול/כתום - יתרה (עודף/גירעון)
